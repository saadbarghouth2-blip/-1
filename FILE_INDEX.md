## 🔄 تدفق العمل

```
المستخدم يفتح الموقع
        ↓
Service Worker ينسجل (src/main.tsx)
        ↓
PWA utilities تعمل (src/lib/pwa.ts) ✅ موجود
        ↓
InstallButton يفحص: هل التطبيق مثبت؟
        ↓
إذا لم يكن مثبتاً:
  ├─ Android: يظهر نافذة تثبيت تلقائية
  └─ iOS: يظهر زر + دليل يدوي
        ↓
المستخدم ينقر للتثبيت
        ↓
التطبيق يتم تثبيته على الشاشة الرئيسية ✅
```

## 🚀 كيفية الاستخدام

### للمطورين:

#### 1. **تطوير محلي:**
```bash
npm run dev
# ثم افتح http://localhost:5173 على الموبايل أو Chrome DevTools
```

#### 2. **بناء الإنتاج:**
```bash
npm run build
npm run preview  # معاينة الإنتاج
```

#### 3. **النشر:**
```bash
# انسخ المجلد dist/ إلى الخادم
# تأكد من HTTPS وأنت جاهز!
```

### للمستخدمين:

#### Android:
1. افتح على Chrome
2. اضغط "ثبّت الآن"
3. تم! 🎉

#### iOS:
1. افتح على Safari
2. اضغط زر التحميل
3. ثم اضغط المشاركة + "إضافة إلى الشاشة الرئيسية"
4. تم! 🎉

## 🐛 استكشاف الأخطاء

### المشكلة: لا يظهر زر التحميل
**الحلول:**
- تحقق من المتصفح (Chrome/Edge الحديث)
- امسح Cache وحاول مرة أخرى
- تأكد من HTTPS

### المشكلة: Manifest لا يتحمل
**الحلول:**
- تأكد من المسار الصحيح `/site.webmanifest`
- تحقق من JSON validity
- امسح Service Worker وأعد تحميل

### المشكلة: الأيقونة لا تظهر
**الحلول:**
- تحقق من وجود الأيقونة في `/icons/`
- تأكد من الأحجام الصحيحة (192x192, 512x512)
- تأكد من صيغة PNG
