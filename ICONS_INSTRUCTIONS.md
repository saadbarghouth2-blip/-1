# App Icon & Art Assets Instructions

## App Icon Requirements
- App Store: 1024 × 1024 px PNG (no transparency)
- iOS App Icons (in app asset catalog) will be generated from a 1024 image
- Adaptive icons for Android: foreground (432×432) and background (1080×1080)

## Apple Touch Icon
- Provide a 180 × 180 px PNG for legacy support

## Maskable icons
- Provide a maskable 512 × 512 PNG for PWA (already in public/icons)

## Recommended style
- Use brand colors: #153b66 (primary)
- Keep simple: logo mark centered on solid background
- Avoid text inside icon if possible (use logomark)

## Filenames suggestion
- icon-1024x1024.png
- icon-180x180.png
- adaptive-foreground.png
- adaptive-background.png

## Where to place icons in repo:
- mobile/assets/icon-1024x1024.png -> used by Expo (app.config.ts icon)
- public/icons/... -> used by PWA and web
