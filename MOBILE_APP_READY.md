## 📲 كيفية الاستخدام

### للمستخدم (الزبون):

#### على Android:
1. افتح الموقع على Chrome أو Edge
2. انتظر نافذة "ثبّت التطبيق"
3. اضغط "ثبّت الآن"
4. تم! التطبيق على الشاشة الرئيسية

#### على iPhone/iPad:
1. افتح الموقع على Safari
2. اضغط المشاركة ⬆️
3. اختر "إضافة إلى الشاشة الرئيسية"
4. أضف الاسم واضغط إضافة
5. تم! التطبيق جاهز

## 🔧 التفاصيل التقنية

### الملفات المحدثة:

#### 1. `public/site.webmanifest`
```json
{
  "name": "متجر ريق - Riq Store",
  "display": "standalone",  // تطبيق كامل
  "orientation": "portrait-primary",
  "icons": [...],  // أيقونات بأحجام مختلفة
  "shortcuts": [...]  // اختصارات سريعة
}
```

#### 2. `index.html`
```html
<!-- PWA Meta Tags -->
<meta name="mobile-web-app-capable" content="yes" />
<meta name="apple-mobile-web-app-capable" content="yes" />
<meta name="apple-mobile-web-app-status-bar-style" content="black-translucent" />
<link rel="manifest" href="/site.webmanifest" />
```

#### 3. `src/components/InstallButton.tsx` (جديد)
مكون جميل يعرض:
- نافذة التثبيت الحديثة
- تعليمات iOS بالعربية
- زر عائم للوصول السريع
- روابط للتطبيقات الأصلية (إن وجدت)

#### 4. `src/main.tsx`
```typescript
import { registerPwaServiceWorker } from './lib/pwa';
void registerPwaServiceWorker();
```
- تسجيل Service Worker
- تفعيل العمل بدون إنترنت

## 🚀 كيفية النشر

المشروع جاهز للنشر الآن! 

### للنشر على الإنترنت:

1. **تأكد من HTTPS**
   - PWA يتطلب HTTPS دائماً
   - معظم منصات الاستضافة توفرها

2. **ابني المشروع**
   ```bash
   npm run build
   ```

3. **انسخ المجلد `dist/`**
   - إلى خادم الويب
   - إلى Netlify/Vercel/أي منصة

4. **تحقق من الملفات المهمة:**
   - ✅ `/site.webmanifest`
   - ✅ `/sw.js`
   - ✅ `/icons/`
   - ✅ `index.html`

## 🧪 الاختبار

### على متصفح سطح المكتب:
```bash
npm run dev  # تطوير محلي
npm run build && npm run preview  # معاينة الإنتاج
```

### على الموبايل الحقيقي:
1. انشر على `localhost` مع HTTPS (أو استخدم ngrok)
2. افتح من هاتفك
3. جرب الزر الزرق "تحميل"
4. تأكد من الأيقونة على الشاشة الرئيسية

### تفعيل DevTools على الموبايل:
- Android: Chrome DevTools (remote debugging)
- iOS: Safari Developer Tools (المفضل)
