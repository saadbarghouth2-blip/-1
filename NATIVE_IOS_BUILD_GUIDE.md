# 📱 دليل شامل: بناء تطبيق iOS الأصلي لمتجر ريق

## 🎯 الهدف
بناء تطبيق **iOS أصلي** لمتجر ريق على **App Store** بدون تعقيدات.

---

## ⚡ الخطوات السريعة (للمستعجلين)

```bash
# 1. الذهاب لمشروع Mobile
cd mobile

# 2. بناء التطبيق الأصلي
npm run build:ios

# 3. الاختبار (اختياري)
npm run preview:ios

# 4. النشر
npm run submit:ios
```

**الآن التطبيق على App Store!** ✅

---

## 📋 المتطلبات

### ✅ لديك بالفعل:
- ✅ Expo.io account
- ✅ Apple Developer account (اختياري الآن)
- ✅ EAS CLI مثبتة
- ✅ app.config.ts معدة

### ⚠️ قد تحتاج لإضافة:
- Apple Certificate (أول مرة فقط)
- Provisioning Profile
- Signing certificate

## 🔐 المرحلة 1: الإعداد الأول (مرة واحدة فقط)

### الخطوة 1: إنشاء حساب EAS
```bash
cd mobile
eas login
# أدخل بيانات حسابك على Expo
```

### الخطوة 2: تهيئة المشروع
```bash
eas project:init
# سيسأل عن معلومات المشروع
```

### الخطوة 3: إعداد Apple Certificate
```bash
eas credentials
# اختر:
# 1. iOS
# 2. Production
# اترك EAS تدير الشهادات تلقائياً ✅
```

## 🔨 المرحلة 2: بناء التطبيق الأول

### الخطوة 1: بناء المشروع

```bash
cd mobile
npm run build:ios
```

أو يدوياً:
```bash
eas build --platform ios --auto-submit
```

### ماذا يحدث:
1. ✅ EAS يبني التطبيق
2. ✅ يوقع الشهادات تلقائياً
3. ✅ ينقله للـ App Store Connect
4. ✅ ينتظر الموافقة (1-3 أيام عادة)

### الخطوة 2: متابعة البناء

```bash
# شاهد حالة البناء
eas build:list

# أو من الموقع
# https://expo.io/dashboard
```

## 📊 المرحلة 3: إعدادات App Store

### ما تحتاج أن تفعله يدوياً:

#### 1. **إنشص صفحة التطبيق على App Store**

اذهب لـ **App Store Connect**:
```
https://appstoreconnect.apple.com
```

أضف تفاصيل التطبيق:
- ✅ اسم التطبيق: "متجر ريق" أو "Riq Store"
- ✅ الوصف (نسختين: إنجليزي + عربي)
- ✅ الكلمات الدلالية (Keywords)
- ✅ الأيقونات (1024×1024)
- ✅ لقطات الشاشة (2-5 صور)
- ✅ تصنيف العمر (4+)
- ✅ معلومات الاتصال

#### 2. **أسعار وتوزيع**

```
السعر: مجاني ✅
الدول: السعودية + دول الخليج
```

#### 3. **سياسة الخصوصية**

أضف رابط سياسة الخصوصية:
```
https://yoursite.com/privacy
```

## 🎯 ملفات مهمة

```typescript

  name: 'Riq Store',              // اسم التطبيق
```

## 🐛 استكشاف الأخطاء الشائعة

### المشكلة: "Bundle ID already exists"
**الحل:**
```
غيّر bundleIdentifier في app.config.ts
من: com.riq.store
إلى: com.riq.store.v2
```

### المشكلة: "Certificate not found"
**الحل:**
```bash
eas credentials
# اختر: Revoke & create new
```

### المشكلة: "Build failed"
**الحل:**
```bash
# نظّف وأعد المحاولة
rm -rf node_modules
npm install
npm run build:ios
```

### المشكلة: "App Store reject"
**الحل:**
- اقرأ ملاحظات Apple
- صحح المشاكل
- أعد الرفع
```bash
npm run build:ios --auto-submit
```
