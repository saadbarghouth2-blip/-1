# ⚡ الخطوات السريعة: بناء تطبيق iOS في 5 دقائق

## 🎯 الهدف
تطبيق iOS أصلي على App Store بدون تعقيدات.

---

## ✅ الخطوة 1: التحضير (مرة واحدة)

```bash
# 1. اذهب للمجلد
cd mobile

# 2. الدخول (لو أول مرة)
eas login

# 3. الإعداد
eas project:init

# 4. الشهادات (اترك EAS تتولاها)
eas credentials
```

**خلص!** الجزء الممل انتهى.

---

## 🚀 الخطوة 2: البناء (كل مرة تحديث)

```bash
# أمر واحد فقط:
cd mobile && npm run build:ios
```

**يبدأ البناء تلقائياً!** ✅

**انتظر 15-20 دقيقة...**

---

## 🧪 الخطوة 3: الاختبار (اختياري)

شاهد تقدم البناء:

```bash
# افتح الرابط:
https://expo.io/dashboard
```

أو من Terminal:

```bash
eas build:list
```

---

## 📤 الخطوة 4: النشر

```bash
# شغّل هذا بعد ما تخلص البناء:
cd mobile && npm run submit:ios
```

**بس كده!** التطبيق راح للـ App Store تلقائياً. ✅

---

## ⏱️ إنتظر...

- **iOS**: 1-3 أيام (Apple تراجع)
- بعدها: **مباشر على App Store**

---

## 🔄 التحديثات المستقبلية

```bash
# غيّر رقم الإصدار
# في: mobile/app.config.ts
# من: version: '1.0.0'
# إلى: version: '1.0.1'

# ثم شغّل:
cd mobile && npm run build:ios
# انتظر انتهاء البناء
cd mobile && npm run submit:ios
```

**خلص التحديث!** ✅

## 🐛 مشكلة؟
